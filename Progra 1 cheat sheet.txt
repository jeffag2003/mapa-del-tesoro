.upper() pasa lo que escribe el usuario a mayuscula
Ej: La entrada es 'SI' y escribe si, sI o Si lo pasa a SI

.lower() pasa lo que escribe el usario a minuscula
Ej: La entrada es 'si' y escribe SI, Si o sI lo pasa a si

input pregunte cosas al usuario (siempre devuelve strings)
Ej: opcion = input ('Digite una opcion:')
    Va a escribir = Digite una opcion:



ambos deben estar dentro de un ciclo (while o for)
break: termina la ejecucion solo de un ciclo (cierra el ciclo de su while)
continue: salta a la siguiente iteracion de un ciclo (se devuelve al inicio de su while)



vectores: son listas


expresiones generadoras
si a la expresion se le aplica alguna funcion (sum,max,min,any,all,etc)
la funcion debe usarse con ()
esto devuelve un num por ejemplo ()
esto devuelve una lista []



all(expresion booleana for elemento in lista/vector)
all devuelve true o false
la expresion booleana puede ser num%2 == 0 o num < 3, etc.

any(expresion booleana for elemento in lista/vector)
any es igual al all, any verifica si alguno es true o false 
all verifica sin todos son true o false, de igual manera recorre la lista
